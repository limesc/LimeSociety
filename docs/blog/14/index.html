대한대장항문학회 제 52차 학술대회 부스 참여

Apr 5, 2019

2019년 4월 5일(금)부터 2019년 4월 7일(일)까지 제주도에서 열린 대한대장항문학회 제 52차 학술대회에 부스로 참석하였습니다.

라임소사이어티는 2018년부터 대한대장항문학회와 Colorectal Cancer Web Database 구축 사업을 진행하고 있습니다. 현재 1차년도 사업을 마무리해가는 과정에 있습니다.

 
 

500명이 넘는 분들이 사전예약하셨으며, 성공적으로 마무리되었습니다.

 
 

감사합니다.
